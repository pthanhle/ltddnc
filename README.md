# Flutter Weather App

Đây là một dự án ứng dụng thời tiết được xây dựng bằng Flutter. Ứng dụng cho phép người dùng xem thông tin thời tiết hiện tại và dự báo cho các ngày tiếp theo dựa trên vị trí của họ.

## Tính năng

*   Xem thông tin thời tiết hiện tại (nhiệt độ, độ ẩm, sức gió, v.v.).
*   Dự báo thời tiết cho nhiều ngày tới.
*   Tự động phát hiện vị trí người dùng.
*   Tìm kiếm thời tiết theo thành phố.
*   Giao diện người dùng đẹp mắt và hiện đại.

## Thư viện sử dụng

*   **Provider:** Quản lý trạng thái.
*   **HTTP:** Gọi API thời tiết.
*   **Geolocator:** Lấy vị trí hiện tại của người dùng.
*   **Flutter Map:** Hiển thị bản đồ.
*   **FL Chart:** Hiển thị biểu đồ thời tiết.
*   **Lottie:** Thêm các hiệu ứng animation đẹp mắt.
*   **Google Fonts:** Sử dụng font chữ từ Google.
*   Và một số thư viện hữu ích khác.

## Bắt đầu

Để chạy dự án này, bạn cần cài đặt Flutter trên máy của mình. Truy cập [trang web của Flutter](https://flutter.dev/docs/get-started/install) để xem hướng dẫn chi tiết.

### Cài đặt

1.  Clone a repository:

    ```bash
    git clone https://your-repository-url.git
    ```

2.  Di chuyển vào thư mục dự án:

    ```bash
    cd your-project-folder
    ```

3.  Cài đặt các dependency:

    ```bash
    flutter pub get
    ```

### Chạy ứng dụng

```bash
flutter run
```
